# 🌸 Dokka Seethamma Tribute Page

A premium tribute webpage dedicated to **Dokka Seethamma**, a legendary philanthropist from Andhra Pradesh who dedicated her life to serving food for hungry travelers and people in need.

This project is designed to showcase her inspiring journey, kindness, and timeless legacy through a modern storytelling website.

---

## 🌿 About The Project

The **Dokka Seethamma Tribute Page** is a frontend web project created using HTML, CSS, and JavaScript.

The main aim of this project is to present her life story in an elegant and interactive way while highlighting values like compassion, humanity, and selfless service.

---

## ✨ Features

- 📌 Responsive modern design
- 🌸 Hero section with inspirational introduction
- 📖 Biography and life story section
- 🕰️ Interactive timeline of her journey
- ❤️ Values and achievements cards
- 🌿 Legacy section
- ✨ Smooth scrolling animations
- 📱 Mobile-friendly layout

---

## 🛠️ Technologies Used

## 🎨 Design Concept

The website follows a traditional heritage-inspired theme with:

- Warm colors
- Elegant typography
- Storytelling sections
- Smooth animations
- Clean user interface

The design represents the simplicity, kindness, and cultural values associated with Dokka Seethamma.
