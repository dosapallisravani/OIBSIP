# 🌸 Luméra - Luxury Skincare Landing Page

## 📖 About the Project  
Luméra is a modern, elegant luxury skincare landing page created as part of the Oasis Infobyte Web Development Internship.  
It is designed to showcase premium skincare products with a clean, responsive, and visually appealing UI.

---

## ✨ Features
- 📱 Fully Responsive Navigation Bar  
- 🖼️ Hero Section with premium UI  
- 🌿 About Luméra section  
- 💎 Featured Products showcase  
- ⭐ Why Choose Us section  
- 💬 Customer Reviews section  
- 📞 Contact section  
- 🔻 Professional Footer  
- 📐 Fully Responsive Design (Mobile + Desktop)

---

- HTML5  
- CSS3
