
# 💼 Personal Portfolio Website

## 📖 About the Project

This is a modern and responsive Personal Portfolio Website developed as part of the **Oasis Infobyte Web Development Internship - Level 1 Task 2**.

The portfolio showcases my skills, projects, education, and contact information with a premium and elegant user interface. It is designed using HTML5, CSS3, and JavaScript while following responsive web design principles.

---

## ✨ Features

- Responsive Navigation Bar
- Premium Hero Section
- About Me Section
- Skills Section
- Projects Showcase
- Education Section
- Contact Section
- Download Resume Button
- Smooth Scrolling
- Typing Animation
- Scroll Reveal Animation
- Scroll To Top Button
- Responsive Hamburger Menu
- Loading Animation
- Floating Background Shapes
- Custom Cursor Effect
- Glassmorphism Design
- Fully Responsive Layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts
- Font Awesome
