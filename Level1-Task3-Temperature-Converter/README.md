# 🌡️ Level1-Task3-Temperature-Converter

A modern and responsive **Temperature Converter Web Application** developed as part of the **Oasis Infobyte Web Development Internship – Level 1 Task 3**.

## 🚀 Live Demo

https://dosapallisravani.github.io/OIBSIP/Level1-Task3-Temperature-Converter/

## ✨ Features

- 🌡️ Convert Celsius, Fahrenheit and Kelvin
- ⚡ Instant temperature conversion
- ✅ Accurate calculations
- ❌ Invalid input validation
- 🧊 Absolute Zero validation
- 🔄 Reset functionality
- 🎨 Premium Glassmorphism UI
- 📱 Fully Responsive Design
- 💻 Built using HTML, CSS and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
